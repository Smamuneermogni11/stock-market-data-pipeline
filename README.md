# 📊 Stock Market Data Pipeline

## 🧠 Overview
This project builds an end-to-end data pipeline that collects stock market data using APIs and prepares it for analysis using SQL and Python.

---

## ⚙️ Pipeline Flow

Stock API → Python Ingestion → Raw Data → Database → SQL Analysis → Notebook Insights

---

## 📦 Project Structure

- scripts/ → Data ingestion and processing scripts
- notebooks/ → Data analysis and visualization
- data/ → Raw and processed datasets
- database/ → SQLite database (if used)

---

## 🚀 Goal
To simulate a real-world data engineering pipeline using stock market data.

---

## 🛠️ Tech Stack
- Python
- Pandas
- yfinance
- SQLite
- Jupyter Notebook

---

## 📈 Future Improvements
- Add macroeconomic data (interest rates, inflation)
